Song maintenance

Batch jobs for similarity and duplicate uploads. Admin only.

Mark similar songs (batch)

For each song in scope (see below), compares it to others in the same language/version-group pool, then merges version groups, creates song links, or applies near-duplicate rules (including moving collection memberships where applicable), using the server similarity thresholds. Scope: the selected language, or all languages if you chose All.

Public songs only Non-public songs only Near-duplicate transfer/delete cutoff date Non-public "loser" songs are only moved/deleted when created on or after this date (server local calendar day).

This can take a long time. The browser tab may look frozen until the server responds—do not refresh.

Remove duplicate uploads

Permanently deletes uploaded songs that are already soft-deleted in the review queue when they match an existing song, have no song-list entries, and are not part of any song collection—same rules as the server cleanup job. Scope follows the language selector above (one language or all).
