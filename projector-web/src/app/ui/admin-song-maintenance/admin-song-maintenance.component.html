Song maintenance

Batch jobs for similarity and duplicate uploads. Admin only.

Mark similar songs (batch)

Scans all languages, compares each public song to others, then merges version groups, creates song links, or applies near-duplicate rules (including moving collection memberships where applicable), using the server similarity thresholds.

This can take a long time. The browser tab may look frozen until the server responds—do not refresh.

Remove duplicate uploads

Permanently deletes uploaded songs that are already soft-deleted in the review queue when they match an existing song, have no song-list entries, and are not part of any song collection—same rules as the server cleanup job.
